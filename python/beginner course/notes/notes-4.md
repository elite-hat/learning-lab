# Pypi

A lot of packages are published by Python Developers on [Python Package Index](https://www.pypi.org)

So, you don't have to write all the code by yourself.|

**Important:** You don't need ot know every package, but you should know some of the most important ones to save your time.

A lot of them are still in development.

In this tutorial, I'm gonna show you how to install packages.

## How to install Packages?

Search: **openpyxl**
Copy the command: `pip install openpyxl`
Paste it into the terminal.

And your package is installed.

## how to use it?

You can use its modules simply as you use the modules written in by you in your own package.
